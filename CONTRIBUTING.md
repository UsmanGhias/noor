# Contributing to Noor ul Haya

Thank you for helping improve Noor ul Haya for Muslims around the world. This project is open to everyone who wants to improve the Quran Majeed and namaz experience with care, accuracy, and respect.

## Open Source Day

This repository was opened for community collaboration on July 14, 2026, Open Source Day for this project.

## How to contribute

1. Fork the repository.
2. Create a focused feature branch.
3. Run formatting, analysis, and tests before opening a pull request.
4. Open a pull request with a clear summary, screenshots for UI changes, and testing notes.
5. Respond respectfully to review feedback.

## Contribution rules

- Keep Islamic content accurate, respectful, and source-backed.
- Do not add Quran text, hadith text, translations, tafsir, namaz guidance, or prayer calculation changes without citing reliable sources in the pull request.
- Keep changes focused. Small pull requests are easier to review and merge.
- Do not include secrets, signing keys, Play Store credentials, private analytics keys, or user data.
- Do not add intrusive tracking, dark patterns, or monetization that harms the spiritual purpose of the app.
- Preserve accessibility, localization readiness, and offline reliability when possible.
- Follow the existing Flutter, Riverpod, GoRouter, and Clean Architecture structure.

## Play Store status

Noor ul Haya is already published on Google Play. Community contributions may be used to improve the published app after review, testing, and release approval by the maintainers.

## Pull request checklist

- [ ] I ran `flutter analyze --fatal-infos`.
- [ ] I ran `flutter test`.
- [ ] I added or updated tests when needed.
- [ ] I added screenshots for visible UI changes.
- [ ] I cited reliable sources for Islamic content changes.
- [ ] I confirmed that no secrets or private files are included.
